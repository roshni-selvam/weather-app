# 🌤️ Weather App

A real-time weather application built with React and OpenWeather API.

## Features
- 🌡️ Live temperature, humidity & weather conditions
- 🔍 Search any city worldwide
- 🎨 Dynamic weather backgrounds

## Tech Stack
- React + Vite
- OpenWeather API
- CSS

## Getting Started
```bash
npm install
npm run dev
```
